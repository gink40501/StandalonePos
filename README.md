單機進出貨銷售的POS系統
===
## [超連結 架構總覽](LOGIC.md)
## 專案簡介
>本專案旨在開發一套單機版的進出貨銷售 POS系統，主要用於中小型零售商店、
玩具店或五金行等實體店鋪，協助商家高效管理商品庫存、進出貨流程與銷售記錄，提升營運效率並
降低人為錯誤。
## 主要功能
> 商品管理

>> 建立、編輯商品資料，支援分類、單位、條碼等資訊。

> 庫存管理

>> 記錄商品進貨與出貨數量，自動更新庫存狀態。

> 銷售作業

>>提供直覺化的銷售介面，支援快速結帳。

> 交易查詢與報表 (未完成)

>>可查詢銷售紀錄、進出貨紀錄，並匯出報表供分析與備份使用。

>資料儲存

>>所有資料皆儲存在本機檔案中，無需依賴資料庫或網路連線即可運作，確保操作簡便且資料完全由使用者掌控。

## 技術架構
> 開發語言:C# 6.0

> 資料儲存：本地 JSON 檔案

> 系統架構：單機版應用，參考MVC架構設計

## 適用場景
> 無需雲端功能或網路環境的小型店家

> 僅需單機操作的銷售與庫存管理需求

> 簡單、穩定、易於維護的系統需求者
